# Abdul Hadi - Software Engineer Portfolio

This repository contains the source code for my personal portfolio website.

## Tech Stack
- **HTML5** & **CSS3** (Vanilla, with Custom Variables for Theming)
- **JavaScript** (ES6+)
- **WebGL / Canvas** (Custom Node Network Animation)
- **Anime.js** (Micro-animations and scroll reveals)

## Features
- 🌗 **Multi-Theme System**: Easily switch between Nebula, Terminal Teal, Amber Circuit, and Arctic themes.
- ⚡ **High Performance**: Vanilla JS and lightweight CSS architecture.
- 🎨 **Dynamic Interactions**: Magnetic buttons, typing effects, custom scrollbars, and hover reveals.
- 📱 **Responsive Design**: Flawless experience across desktop, tablet, and mobile devices.

## Setup
Simply clone the repository and open `index.html` in any modern web browser. No build steps required.
